# SolarUI

A lightweight UI library built for OpenGL / FreeGLUT.

## Features

- Buttons
- Sliders
- Labels
- Tiles

## Installation

Link against:

libSolarUI.a

Include:

#include "SolarUI.h"

## Example

```cpp
SolarUI::Button btn(50, 50, 120, 40, "Hello");
btn.Draw();
```
