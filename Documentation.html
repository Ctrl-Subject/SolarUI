Overview

SolarUI is a lightweight user interface library designed for FreeGLUT/OpenGL applications. It provides a small collection of UI controls while remaining easy to integrate into existing projects.

Features

  • Minimal dependencies
  • Bitmap font rendering
  • Simple mouse input handling
  • Labels
  • Buttons
  • Sliders

Getting Started

Typical application flow:

#include "SolarUI.h"

int main()
{
    SolarUI::Init();

    while (true)
    {
        SolarUI::Update();
        SolarUI::DrawFrame();
    }
}

Global Variables

Name Description
MouseX Current mouse X position.
MouseY Current mouse Y position.
MouseDown True while the mouse button is held.
MousePressed True only during the frame the button was pressed.
CurrentFont Currently selected GLUT bitmap font.

SetFont()

SolarUI::SetFont(GLUT_BITMAP_HELVETICA_18);
Changes the active bitmap font used when rendering text.

Core Functions

Init()

Initialises the SolarUI framework and prepares it for use. Call this once before creating or rendering any UI.

Update()

Updates SolarUI for the current frame. Processes mouse input and updates interactive controls.

DrawFrame()

Draws the current user interface. Call once every frame after Update().

Label

Displays static text.

Declaration

SolarUI::Label title("Hello", 20, 20);

Members

Member Description
Text Displayed string.
X Horizontal position.
Y Vertical position.

Functions

  • Draw() — Renders the label.

Button

Represents a clickable button.

Declaration

SolarUI::Button play(20,20,150,40,"Play");

Members

Member Description
XHorizontal position.
YVertical position.
WWidth.
HHeight.
TitleButton label.

Functions

  • Draw() — Draws the button.
  • Contains() — Returns whether a point lies inside the button.
  • IsClicked() — Returns true if clicked this frame.

Slider

Represents a horizontal slider with a value from 0.0 to 1.0.

Declaration

Functions

  • Update() — Updates the slider value.
  • Contains() — Returns whether a point lies inside the slider.
